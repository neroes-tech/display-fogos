# Neroes — Display Fogos

Painel de prioridades estratégicas da Neroes, deployado no Vercel e aberto em modo fullscreen num monitor Android ELO.

## Stack

- HTML + CSS + JS vanilla (sem dependências)
- Google Fonts: Syne, Space Mono, Inter
- Deploy: Vercel (static)

## Estrutura

```
display-fogos/
├── index.html
├── vercel.json
└── assets/
    ├── neroes_favicon.png
    ├── neroes_logo.png
    └── favicon.ico
```

## Deploy

O projeto está configurado para deploy automático no Vercel a partir do repositório `neroes-tech/display-fogos`. Qualquer push para `main` despoleta um novo deploy.
